    <h2>⚡ Getting Started</h2>
    <br />
    <h4>Config BD con Prisma</h4>
    <p>Una vez configurada la URL de la base de datos para conectar Prisma, realiza los siguientes pasos:</p>
    <pre>
    <br />
    <h4> Crea tablas en BD</h4>
    ```bash
    pnpm prisma migrate dev --name init
    ``` 
    <h4> Actualiza tablas en BD</h4>
    ```bash
    pnpm prisma migrate dev --name update
    ```

    <h4> Generate tables cada que se hacen cambios al esquema</h4> 
    ```bash
    pnpm prisma generate
    ```

    <h4> Reset prisma si requieres borrar toda la BD de cero</h4> 
    ```bash
    pnpm prisma db push --force-reset
    ```

    <h4> Run seed</h4> 
    ```bash
    pnpm run seed
    ```
    </pre>
    <p>Convierte el archivo <code>.env.example</code> en <code>.env</code> y actualiza los valores de las variables de ambiente.</p>
    <p>Finalmente, corre los siguientes comandos para iniciar el servidor de desarrollo:</p>
    <pre>
    ```bash
    pnpm install
    pnpm run dev
    ```
    </pre>
    <p>Ahora podrás ver la aplicación lanzada en <a href="https://what2eat-dashboard.vercel.app/login">https://what2eat-dashboard.vercel.app</a>.</p>
